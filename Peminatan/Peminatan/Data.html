Data Courses & Mini Projects

1. Computer Vision

Learn how computers can "see" and understand images or videos, including object detection, face recognition, and segmentation.

Mini Project Ideas:
  • Real-time object detection using YOLOv5
  • Automatic attendance system using face recognition
  • Facial expression analysis for emotion detection

2. Applied Artificial Intelligence

Focuses on applying AI algorithms in various domains such as recommendation systems, optimization, and classification.

Mini Project Ideas:
  • AI-based movie recommendation system
  • Customer service chatbot using NLP
  • Route optimization for delivery using genetic algorithms

3. Big Data Analytics

Learn how to process and analyze large volumes of data using technologies like Hadoop and Spark.

Mini Project Ideas:
  • Twitter hashtag trend analysis using Apache Spark
  • Interactive dashboard for e-commerce data visualization
  • Big data ETL pipeline using Hadoop and Hive

4. Cloud Databases

Discusses distributed and scalable database management using cloud platforms like AWS, Azure, or GCP.

Mini Project Ideas:
  • Building a REST API with Firebase Realtime Database as backend
  • Product data management using Cloud SQL and App Engine
  • Integrating MongoDB Atlas with a Node.js application

5. Data Centers

Focuses on infrastructure, management, and security of large-scale data centers to support modern information systems.

Mini Project Ideas:
  • Simulating data center network design with Cisco Packet Tracer
  • Server performance monitoring using Zabbix or Prometheus
  • Needs analysis and infrastructure design for a university data center
