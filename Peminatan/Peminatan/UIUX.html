UI/UX Courses & Mini Projects

1. Serious Game

Explore games designed for learning, health, or social change purposes.

Mini Project Ideas:
  • Develop an educational game for language learning
  • Create a mental health awareness game using storytelling
  • Build a simulation game for disaster preparedness

2. Prototyping

Learn how to create interactive prototypes for apps and websites.

Mini Project Ideas:
  • Design a mobile banking app prototype with Figma
  • Create an interactive travel booking website prototype
  • Build a usability-focused e-commerce checkout flow

3. Multimedia Applications

Combine visuals, audio, and interactivity to create digital experiences.

Mini Project Ideas:
  • Create a multimedia portfolio website with animations
  • Design an interactive storytelling web app
  • Build a digital art exhibition experience using 360Â° media

4. Game Design

Learn the fundamentals of designing engaging game mechanics and stories.

Mini Project Ideas:
  • Design a game concept document with levels and characters
  • Create a prototype of a puzzle game using Godot or Unity
  • Develop a narrative-driven game using Twine or RPG Maker

5. User Evaluation

Study how users interact with systems and improve UX through feedback.

Mini Project Ideas:
  • Conduct a usability test on a student-designed app
  • Analyze user experience data and report improvement suggestions
  • Run an A/B test on two versions of a UI design
